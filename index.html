About Me

I began building web experiences in 2020 while supporting students at Taqdis Computers. What started as filling e-Seva forms turned into writing HTML, CSS, and PHP from scratch, extracting data from public PDFs, and deploying tools on free hosting — all without a personal computer at home.

Today I ship full-stack products using Next.js, Node.js, MongoDB, and MySQL, deploy using Vercel, Cloudflare, and cPanel, and harden them through ethical hacking practice on TryHackMe. I like combining DIY resourcefulness with simple, usable UIs.

Experience: Taqdis Computers • Computer Operator (Jan 2020 – Jul 2025)
  • Worked at a busy cyber/DTP shop handling online government forms (PAN, passport, caste/domicle, e-seva), DTP tasks, and client support.
  • Self-taught web development on the job: HTML/CSS, PHP, MySQL, WordPress and basic Node.js — built internal tools (landing pages, quiz system, CMS) used by shop customers and students.
  • Managed device metadata logging, lightweight security hardening for admin panels, and learned hosting & DNS (cPanel, free hosts, Cloudflare).

Highlighted Projects

Some selected projects across Next.js, PHP, and Node.js:

1. i-Card Generator

Next.js + MongoDB workflow that lets teachers upload student data, generate QR-enabled A3 print layouts, and share login-secured downloads using JWT and Cloudinary.

Stack: Next.js, MongoDB, Cloudinary, JWT, Bootstrap
Live: https://icard-generator.intesab.live/

2. Quiz Platform for Students

Gamified quiz system for Taqdis Computers students with instant answer feedback, leaderboards, and attempt history secured behind role-based access.
